# CELPIP AI Coach - Visual & Funnel Improvements Plan

## Current State Analysis

### What's Working
- Dark premium theme with good color palette
- Skill cards with color coding
- Pain/Solution section with good copy
- Express Entry emotional hook
- Mobile sticky CTA

### Critical Issues Identified

#### 1. VISUAL - Hero Section Lacks Impact
- No visual proof (screenshots, demo, video placeholder)
- Hero feels text-heavy without visual hierarchy break
- Missing animated counter/stats to build credibility
- No gradient hero background (defined in design system but unused)

#### 2. VISUAL - Monotonous Card Design
- All sections use same bg-card (#232733) with same border
- No visual differentiation between sections
- Missing section dividers/decorative elements
- Cards lack depth - no glassmorphism or gradient borders

#### 3. FUNNEL - Weak Social Proof
- "350+ students" is low - needs better framing
- No star ratings on testimonials
- No "as seen on" or trust badges
- Missing live activity indicator ("X people practicing now")
- No before/after score visualization

#### 4. FUNNEL - CTA Optimization
- Only one CTA style throughout (red gradient)
- No secondary CTA for less committed users
- Missing "Try a free exercise" quick demo CTA
- Header has "Sign In" but no prominent "Start Free" button

#### 5. VISUAL - Missing Visual Elements
- No illustrations or icons beyond Lucide
- No animated statistics/counters
- No progress visualization in testimonials
- Missing video/screenshot placeholder in hero

#### 6. FUNNEL - Missing Urgency & Scarcity
- No countdown timer for offers
- No "limited spots" messaging
- No "X people signed up today" counter
- CommunityPopup fires too early (2s) and promotes Telegram instead of conversion

#### 7. VISUAL - Typography Hierarchy
- Section titles all same size/style
- Missing decorative elements (lines, badges above titles)
- No variation in section backgrounds

## Implementation Plan

### Phase 1: Hero Section Overhaul
- Add gradient hero background
- Add animated stats counter bar
- Add "live activity" indicator
- Improve CTA with secondary option
- Add floating mockup/screenshot area

### Phase 2: Section Visual Differentiation
- Alternate section backgrounds
- Add gradient border cards for key sections
- Improve testimonial cards with star ratings and score bars
- Add decorative section dividers

### Phase 3: Funnel Optimization
- Add "Start Free" CTA to header
- Add quick demo CTA in hero
- Improve social proof with live counter
- Add trust badges section
- Optimize CommunityPopup timing and purpose
- Add email capture popup activation

### Phase 4: Micro-interactions & Polish
- Add scroll-triggered animations
- Improve hover states
- Add number counting animations
- Polish responsive design
